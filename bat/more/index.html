Introduction

In this tutorial, we will explore more advanced commands in batch scripting, including conditional statements, loops, and functions.

Conditional Statements

Conditional statements allow batch files to make decisions. Here is an example using the IF statement:


                @echo off
                set /p userInput="Enter a number: "
                if %userInput%==1 (
                    echo You entered one.
                ) else (
                    echo You did not enter one.
                )
                pause

The IF statement checks if the user input is equal to 1. If it is, it displays "You entered one." Otherwise, it displays "You did not enter one."

Loops

Loops are used to repeat a block of commands. Here is an example using the FOR loop:


                @echo off
                for %%i in (1 2 3 4 5) do (
                    echo %%i
                )
                pause

The FOR loop iterates through the list (1 2 3 4 5) and echoes each number to the screen.

Functions and Labels

Batch files can use labels and the GOTO command to create functions and subroutines. Here is an example:


                @echo off
                call :myFunction
                echo Back to main
                pause

                :myFunction
                echo Inside function
                goto :eof

This batch file defines a function named :myFunction. The call command runs the function, and goto :eof returns to the main program.

Examples

Here are some more advanced examples of batch files:

Example 1: File Existence Check


                @echo off
                if exist "C:\example.txt" (
                    echo File exists.
                ) else (
                    echo File does not exist.
                )
                pause

This example checks if a file named "example.txt" exists in the C: drive and displays a message accordingly.

Example 2: Loop Through Files


                @echo off
                for %%f in (C:\*.txt) do (
                    echo %%f
                )
                pause

This example loops through all .txt files in the C: drive and echoes their names.

Example 3: Simple Menu


                @echo off
                :menu
                echo 1. Option 1
                echo 2. Option 2
                echo 3. Exit
                set /p choice="Choose an option: "
                if %choice%==1 goto option1
                if %choice%==2 goto option2
                if %choice%==3 goto exit
                goto menu

                :option1
                echo You chose option 1.
                pause
                goto menu

                :option2
                echo You chose option 2.
                pause
                goto menu

                :exit

This example creates a simple menu that lets the user choose between different options. Depending on the choice, it jumps to different labels.
