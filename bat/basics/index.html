Introduction

A batch file is a script file in DOS, OS/2, and Microsoft Windows. It consists of a series of commands to be executed by the command-line interpreter, stored in a plain text file. Batch files are used to automate repetitive tasks, configure system settings, and run programs.

Syntax

The basic syntax of a batch file is a simple series of commands. Here is an example:


                @echo off
                echo Hello, World!
                pause

Each line in a batch file represents a command to be executed. The @echo off command prevents the commands from being displayed in the command prompt. The echo command prints text to the screen, and the pause command waits for the user to press a key before continuing.

Common Commands

Here are some commonly used commands in batch files:

  • echo: Displays messages, or turns command echoing on or off.
  • pause: Suspends processing of a batch program and displays a message.
  • cls: Clears the screen.
  • rem: Adds comments in the batch file.
  • set: Sets environment variables.
  • if: Performs conditional processing in batch programs.
  • for: Runs a specified command for each file in a set of files.
  • goto: Directs the batch program to jump to a labeled line.
  • call: Calls another batch file and returns to the current one after execution.

Examples

Here are some basic examples of batch files:

Example 1: Display Message


                @echo off
                echo This is a batch file example.
                pause

This example displays a message and waits for the user to press a key.

Example 2: Creating a Directory


                @echo off
                mkdir MyNewFolder
                echo Folder created.
                pause

This example creates a new directory named "MyNewFolder" and confirms the creation.

Example 3: Deleting Files


                @echo off
                del /Q C:\Temp\*.tmp
                echo Temporary files deleted.
                pause

This example deletes all .tmp files in the C:\Temp directory.
