Introduction

Batch files can be used to perform various file and directory operations such as creating, deleting, renaming, and copying files and directories. In this tutorial, we will cover these basic operations.

Common Commands

Here are some commonly used commands for file and directory operations:

  • mkdir: Creates a new directory.
  • rmdir: Removes a directory.
  • del: Deletes one or more files.
  • copy: Copies files from one location to another.
  • move: Moves files from one location to another.
  • ren: Renames a file or directory.

Examples

Here are some examples of file and directory operations:

Example 1: Creating and Removing a Directory


                @echo off
                mkdir C:\MyNewFolder
                echo Directory created.
                pause
                rmdir C:\MyNewFolder
                echo Directory removed.
                pause

This example creates a new directory named "MyNewFolder" in the C: drive and then removes it.

Example 2: Copying and Moving Files


                @echo off
                copy C:\example.txt D:\example.txt
                echo File copied.
                pause
                move D:\example.txt D:\new_example.txt
                echo File moved and renamed.
                pause

This example copies a file from the C: drive to the D: drive and then moves and renames it.

Example 3: Deleting Files


                @echo off
                del C:\Temp\*.tmp
                echo Temporary files deleted.
                pause

This example deletes all .tmp files in the C:\Temp directory.
