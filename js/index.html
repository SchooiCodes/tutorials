Introduction

JavaScript is a versatile programming language that is commonly used in web development to create interactive effects within web browsers.

Syntax

JavaScript syntax is the set of rules that define a correctly structured JavaScript program. Here is an example of JavaScript syntax:


                // This is a single line comment
                var x = 5; // Declare a variable
                var y = 6;
                var z = x + y; // Perform addition
                console.log(z); // Output the result

Examples

Below are some basic examples of JavaScript:

Example 1: Alert Box


                <script>
                    alert("Hello, World!");
                </script>

Example 2: Changing HTML Content


                <script>
                    document.getElementById("demo").innerHTML = "Hello JavaScript!";
                </script>
                <p id="demo"></p>
